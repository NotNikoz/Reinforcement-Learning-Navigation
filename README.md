# Reinforcement-Learning-Navigation
Modelado, simulación y control de un robot camarero mediante Reinforcement Learning en Unity

Este proyecto tiene como objetivo principal el estudio y aplicación de técnicas de
Reinforcement Learning (RL) para la navegación autónoma de un robot móvil camarero tipo
Pudu Bot. Para ellos se hará uso de Unity. Se modelará el robot y se
configurará un escenario de navegación que permita la interacción con un agente de
aprendizaje. El trabajo se centrará en el entrenamiento de políticas basadas en RL (con
algoritmos como rsl_rsl_ppo) para la resolución de tares de navegación, siendo el
desplazamiento hacia metas definidas y evasión de obstáculos el principal objetivo.

Se han creado tres escenarios distintos para comprobar el rendimiento del entrenamiento del robot. 
Se añade además el código de entrenamiento usado junto a material de ayuda.
